ul vs. ol

  • unordered list
    • HTML-tag: ul
    • listet Punkte mit bullet point auf
    • Beispiele:
      • To-Do Liste
      • Einkaufsliste
      • Ideensammlung
    • Weiterführende Links
  • ordered list
    • HTML-tag: ol
    • listet Punkte mit Nummerierung auf
    • Beispiele:
      1. Inhaltsverzeichnis
      2. Startnummern
      3. Gewinnerliste
    • Weiterführende Links
